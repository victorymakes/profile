# Profile Website

This is a personal profile website that showcases my skills,projects, experiences, educations, and more. 
The website is fully responsive and optimized for performance.

# Features

- [x] Responsive design that works on all devices.
- [x] Optimized for performance and SEO.
- [x] Dynamic content loading using Next.js server-side rendering.
- [x] Interactive elements and animations using React Bits.
- [x] Customizable and easy to maintain codebase.

# Tech Stacks

- [Next.js](https://nextjs.org/)
- [Zod](https://zod.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Shadcn](https://ui.shadcn.com/)
- [Magic UI](https://magicui.design/)
- [React Bits](https://reactbits.dev/)

# UI Components

## Goal
The website should feel like it was crafted by a world-class designer, award-winning design: ultra-clean layout, and **luxurious effects**.

## Guideline

- Follow best practices in UI/UX design to ensure a seamless user experience.
- Follow the Shadcn UI design system and guidelines to maintain consistency across the website.
- Leverage shadcn/ui, Magic UI, and React Bits instead of starting from scratch; this speeds development and ensures a consistent, user-friendly UI.
- Never put custom components under the ui folder.
